# SmartFlow Suite 🚀

An enterprise-grade Contract Management Dashboard developed using Flask, SQLite, Bootstrap, and Chart.js with modern animated UI, analytics, authentication, and responsive design.

---

# ✨ Features

- Secure Login Authentication
- Role-Based Dashboard Access
- Add / Delete / Manage Agreements
- PDF Contract Upload System
- Contract Analytics Dashboard
- Revenue Visualization Charts
- Expiry Alerts & Notifications
- Search Functionality
- Export Report Feature
- Dark / Light Theme Toggle
- Modern Glassmorphism UI
- Fully Responsive Design
- Interactive Animations

---

# 🛠️ Tech Stack

## Frontend
- HTML5
- CSS3

- Python
- Flask Framework

- SQLite3

# ⚙️ Installation Steps

## 1️⃣ Clone Repository

```bash
git clone https://github.com/your-username/smartflow-suite.git
```

---

## 2️⃣ Open Project Folder

```bash
cd smartflow-suite
```

---

## 3️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

---

## 4️⃣ Run Flask Server

```bash
python app.py
```
